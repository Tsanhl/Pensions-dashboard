# Pensions Dashboard

A static prototype of a UK pension and savings adequacy dashboard.

The dashboard helps users:

- record DC, DB, hybrid and State Pension details;
- compare projected retirement income with a monthly goal;
- test short-term savings and emergency-buffer pressure;
- review contribution, household, life-event and investment-boundary checks;
- ask a built-in AI-style assistant for plain-English guidance and risk triage.

This is a prototype for public-understanding and policy-design purposes. It provides financial guidance and legal information only, not regulated financial advice or legal advice.
